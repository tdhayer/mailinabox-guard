Postfix (Mail Server)

Configure network protocols used by the mail server. Separating incoming and outgoing protocol preferences can help you maintain IPv6 connectivity while avoiding aggressive IPv6 anti-spam blocklists when sending.

Select whether Postfix should support both IPv4 and IPv6 globally.

Select whether Postfix should prefer a specific protocol when sending outbound mail. "Prefer IPv4" will always use IPv4 if the receiving server supports it, but can fall back to IPv6 if necessary.
